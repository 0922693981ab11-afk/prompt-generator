# 🚀 Roly的 AI 提示詞快速產生器

這是一款專為行銷企劃、設計師與一般大眾打造的「AI 提示詞 (Prompt) 組合工具」。透過直覺的點擊介面，將複雜的提示詞工程簡化，幫助你快速產出精準、高品質的 AI 生成指令！

## ✨ 工具特色

* **⚡ 直覺式標籤組合：** 提供 7 大維度（角色、版型模板、視覺風格、詮釋情境、目標對象、色彩搭配、視覺效果）的預設標籤，點擊即可自動串接成完整提示詞。
* **🤖 內建 AI 視覺分析（黑科技）：** 支援上傳圖片或 PDF（如簡報截圖），系統會自動呼叫 Gemini AI 分析畫面的視覺元素、排版邏輯與配色，並將分析結果無縫轉化為提示詞。
* **📋 即時預覽與一鍵複製：** 右側面板會即時顯示組合好的 Prompt，確認無誤後點擊「建立副本」即可一鍵複製，馬上貼給 ChatGPT、Midjourney 或 Gemini 使用。
* **💾 雲端自動儲存：** 內建 Firebase 支援，即使不小心重整網頁，你剛剛勾選的設定與輸入的自訂筆記也不會消失。

## 💡 實際範例 (Prompt 產出成果)

透過這個工具，你可以輕鬆組合出以下專業等級的提示詞：

### 🎯 範例一：高質感保溫瓶商業攝影
* **選擇組合：** `視覺設計師` + `寫實風格` + `嚴謹客觀` + `經典純白` + `精確裁切置中`
* **自訂需求加入：** 請生成一張頂級不鏽鋼保溫瓶的商業攝影圖。保溫瓶表面帶有細緻的金屬拉絲紋理，瓶身反射出柔和的攝影棚打光。背景需有微微的水滴與冷凝霧氣效果，強調保溫/保冷性能。
* **產出效果：** 能精準生成用於電商首頁、具備高級質感與光影細節的商品主圖。

### 🎯 範例二：1/7 比例公仔模型設計圖
* **選擇組合：** `領域專家` + `三維圖像` + `充滿好奇` + `粉彩馬卡龍` + `預留排版空間`
* **自訂需求加入：** 設計一款 1/7 比例的動漫女角色公仔模型概念圖。角色穿著帶有未來科技感的半透明服裝，姿勢呈現躍動感，底座為具有發光二極管效果的幾何碎塊。請提供清晰的正面與側面細節。
* **產出效果：** 能給 3D 建模師或模型製造廠作為高標準參考圖，材質與比例明確。

### 🎯 範例三：傳統水墨風格賀卡設計
* **選擇組合：** `視覺設計師` + `水彩畫風` + `溫暖同理` + `大地自然` + `手繪標註`
* **自訂需求加入：** 請設計一系列共 56 張的馬年新年祝福賀卡。主體採用傳統水墨暈染技法繪製駿馬，並結合現代 Q 版的圓潤身形比例，呈現既莊重又討喜的節慶氛圍。
* **產出效果：** 完美融合傳統藝術與現代節慶需求，產出風格統一的系列文創圖像。

## 🛠️ 使用方法

1.  **選擇主標題：** 在左側「0. 封面標題」輸入你這次任務的核心名稱（選填）。
2.  **挑選參數：** 依序點擊下方的按鈕，選擇最符合你需求的「角色」、「風格」、「色調」等設定。
3.  **進階分析 (選用)：** 如果你有參考圖，可以點擊上方的「AI 投影片視覺分析」上傳圖片，讓 AI 幫你寫視覺描述。
4.  **補充細節：** 在底部的「8. 自訂需求」框內，用自己的話補充特定要求。
5.  **複製使用：** 確認右側深色面板內的提示詞無誤，點擊右上角「複製」即可！
