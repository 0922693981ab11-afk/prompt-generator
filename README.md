# 🚀 啓仲哥 的 AI 提示詞快速產生器 (Prompt Generator)

這是一款專為**企業管理、工程採購、視覺設計及學術研究**打造的「全方位 AI 指令組合工具」。透過直覺的點擊介面與 AI 視覺分析技術，將複雜的提示詞工程 (Prompt Engineering) 簡化，幫助您產出具備「專家思維」的高品質生成指令！

## ✨ 工具特色

* **⚡ 十一層級維度組合**：提供從「輸出格式」、「專家角色」到「色調細節」等 11 種維度標籤，點擊即可串接成邏輯嚴密的結構化 Prompt。
* **✍️ 靈活自訂擴充**：每個分類下方均設有「自訂輸入框」，若預設按鈕不符需求，可直接輸入特定條件，系統將自動整合進指令中。
* **🤖 AI 視覺與文獻黑科技**：支援上傳 **圖片、截圖或 PDF**。內建 Gemini AI 分析功能，能自動萃取長文重點或拆解圖片視覺元素，並將分析結果無縫餵養給下一個指令。
* **📋 專業級輸出框架**：
    * **格式型**：文章優化、對照表、分類簡介。
    * **餵養型**：長文轉社群貼文、文獻轉商業應用方針。
    * **場景型**：嚴格的角色扮演與語氣限制。
* **💾 雲端即時同步**：整合 Firebase 實時資料庫，您的選取狀態與自訂備註會自動儲存，即使重整網頁資料也不會遺失。

## 💡 實際應用場景 (Prompt 產出成果)

透過此工具，您可以輕鬆組合出專業等級的複合式指令：

### 🎯 場景一：工程採購與風險審核 (專業職人)
* **選擇組合**：`採購合約專家` + `紅隊審核員` + `工程範本健檢` + `嚴苛批判`
* **自訂需求**：於「原始資料區」貼上招標文件草案。
* **產出效果**：AI 會以資深採購視角，一針見血指出合約中的風險死角、工期漏洞與違約責任爭議點。

### 🎯 場景二：碩博士論文/商業文獻萃取 (學術商務)
* **選擇組合**：`策略顧問` + `餵養型：文獻轉商業應用` + `深度分析報告` + `決策高層`
* **功能應用**：上傳 PDF 論文，使用「詳細分析」功能。
* **產出效果**：將艱澀的學術理論轉化為高層看得懂的「商業獲利模式」與「具體執行步驟」。

### 🎯 場景三：高質感商品視覺提案 (設計創意)
* **選擇組合**：`視覺設計大師` + `歐美時尚雜誌` + `寫實風格` + `奢華黑金`
* **自訂需求**：請設計一款高端智能手錶的廣告視覺。
* **產出效果**：生成具備極強視覺張力、符合時尚雜誌排版審美且色彩精確的 Midjourney/DALL-E 指令。

## 🛠️ 使用方法

1.  **AI 視覺/文獻預處理 (選用)**：
    * 點擊上傳參考圖片或 PDF，選擇「詳細分析」或「精簡分析」。分析結果會自動填入下方的原始資料區。
2.  **設定核心主題**：
    * 在「0. 封面標題」輸入專案名稱，這將定義 AI 任務的核心。
3.  **精選專家參數**：
    * 依序點擊 1~11 項標籤。若有特殊需求，直接在該分類的底下的輸入框填寫（例如在語氣區填入：`帶點周星馳電影的幽默感`）。
4.  **餵養原始資料**：
    * 在「12. 原始資料區」貼上您要 AI 處理的長篇文章、專案概要或數據。
5.  **一鍵複製與執行**：
    * 檢查右側藍色面板產出的結構化 Prompt，點擊「建立副本」後貼往 ChatGPT 或 Gemini 即可獲得最佳成果！
